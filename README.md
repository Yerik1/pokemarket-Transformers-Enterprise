# PokéMarket

PokéMarket es una plataforma digital enfocada en el mercado de cartas coleccionables Pokémon, diseñada para centralizar información de precios, historial de referencias y apoyo a procesos de subasta y valoración. Su objetivo es ofrecer datos más confiables y estructurados tanto a coleccionistas individuales como a clientes B2B, reduciendo la dependencia de procesos manuales y mejorando la transparencia del mercado.

El sistema busca transformar un modelo operativo basado en hojas de cálculo y actualizaciones manuales en una solución escalable, con capacidades de consulta, administración de catálogo, gestión de subastas e integración mediante API para servicios externos y comercios especializados.
